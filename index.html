O Crea Jr de São Gotardo está realizando um projeto de cartinhas de Natal para espalhar alegria e esperança para crianças carentes da cidade. Para participar, você pode adotar uma cartinha e escolher o presente que a criança deseja. Ou se preferir, você pode fazer uma doação, via PIX, para ajudar a financiar o projeto. Neste Natal, faça a diferença na vida de uma criança. Adote uma cartinha ou faça uma doação.
